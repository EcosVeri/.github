# EcosVeri

Organization-wide profile, governance, and community standards for EcosVeri.

EcosVeri builds open-source infrastructure for evidence-first AI systems, intelligent retrieval, search, knowledge graphs, and scientific computing.

## Ecosystem

EcosVeri projects focus on practical infrastructure for verifiable AI workflows:

- Evidence-first research assistants.
- Search and retrieval engines for AI agents.
- Knowledge graph and GraphRAG tooling.
- OCR and multimodal retrieval systems.
- Claim verification and evaluation tooling.

## Principles

- Evidence before assertion.
- Reproducible workflows and transparent evaluation.
- Open interfaces that compose across projects.
- Python for product velocity, Rust for performance-critical components, and R for statistical and scientific computing.
