## Mission

EcosVeri develops open-source infrastructure that enables transparent, evidence-first, and reproducible AI systems.

Our projects focus on trustworthy retrieval, verification, scientific computing, and developer tooling.

## Projects

| Project | Focus |
| --- | --- |
| Verilume | Evidence-first AI research assistant |
| Needle | High-performance search engine for AI agents |
| GraphForge | Knowledge graph and GraphRAG toolkit |
| VisionStore | OCR and multimodal retrieval |
| ClaimCheck | Evidence verification |
| EvalBench | Evaluation tools for AI retrieval and reasoning |

## Principles

- Evidence before assertion.
- Reproducible research and transparent evaluation.
- Open interfaces over closed workflows.
- Python for product velocity, Rust for performance-critical systems, and R for statistical and scientific computing.

## Status

EcosVeri is in early formation. Public repositories and documentation will be published as individual projects mature.
