# Supported Versions

EcosVeri is in early formation. Before the first stable releases, support is best-effort and focused on the latest main branch of each repository.

| Stage | Support |
| --- | --- |
| Pre-release | Best-effort fixes on latest main |
| `0.x` releases | Security and critical correctness fixes when practical |
| `1.x` releases | Version-specific support policy to be published per project |
